# 💖 For You — Персональный романтический сайт

Интерактивная «мини-вселенная» — романтический подарок в формате веб-сайта.

## 🚀 Быстрый старт

1. Открой `index.html` в браузере (двойной клик)
2. Или запусти локальный сервер:

```bash
npx serve .
```

## ⚙️ Настройка

Все персональные данные — в **`js/config.js`**:

| Параметр | Описание |
|----------|----------|
| `herName` | Имя девушки |
| `meetDate` | Дата знакомства (сейчас: 4 января 2026, 08:30) |
| `route` | Адреса для блока «GPS» |
| `gallery` | Фото и гифки для галереи |
| `playlist` | Плейлист (mp3 + обложки) |
| `timeCapsule` | Капсула времени (дата + текст) |
| `fromHeart` | Блок с гифкой и эмоциями |

### Фото, гифки, музыка

Все медиафайлы — в папку `assets/`. Полный список с галочками:

**→ [`assets/CHECKLIST.md`](assets/CHECKLIST.md)**

## 📦 Деплой

### GitHub Pages
1. Создай репозиторий на GitHub
2. Залей файлы проекта (**без** `node_modules/` — уже в `.gitignore`)
3. Убедись, что все 16 файлов из чеклиста лежат в `assets/`
4. **Settings → Pages → Branch: `main` → `/ (root)` → Save**
5. Подожди 1–2 минуты, открой `https://ТВОЙ-ЛОГИН.github.io/ИМЯ-РЕПО/`
6. Пройди проверку из чеклиста на телефоне и компьютере

### Netlify
Перетащи папку проекта на [netlify.com/drop](https://app.netlify.com/drop)
